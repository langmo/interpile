General Information

Before you contact us with a question, please read the Documentation to check if your question might already be answered there.

Bugs and Feature Requests

InterPile is hosted on GitHub. As such, the easiest way to report bugs, feature requests, and similar is to create a so called "issue", which you can do here: https://github.com/langmo/interpile/issues

The GitHub "issue" system has the advantage that it is easier to track issues, as well as that other users can comment on these issues. Thus, we prefer issues over e-mails, even though you can still write us an e-mail if you prefer to do so.
